# How to update


基本的には以下のサイトを参考
- https://toha-guides.netlify.app/posts/getting-started/github-pages/
- https://github.com/hugo-toha/toha


## ローカルでの検証
```
hugo server -t toha -w
``` 
http://localhost:1313 にアクセスする

## theme/toha配下での変更

```
git add *
git commit -m 
git push
```

## theme/toha配下以外での変更

```
git add *
git commit -m 
git push origin source
```
